about the data

We are using a data set by the Chicago Police which reflects reported incidents of crime (except murders) occurring in Chicago from 2001 to present. The data set holds updated records minus the most recent seven days, however we have limited our data set to be from 2002 - 2025. More information can be found on the City of Chicago Data Portal.
